title: Integrate
description: High-level guides for connecting to Kaspa, inspecting DAG state, building wallet flows, and operating the node layer behind an integration.
---

Start here when you want the first useful integration steps: connect over RPC, inspect DAG state, manage wallet flows, read accepted transactions, attach a payload, or run your own node.

Begin with [Getting started](/integrate/getting-started) for installation, RPC connection, and a first DAG read. Then use [Wallet](/integrate/wallet) for wallet lifecycle, accounts, and sending. After that, use [Accepted transactions](/integrate/accepted-transactions) when you want to listen for accepted transactions and data, [Transaction payload](/integrate/transaction-payload) when you want to carry application data, and [Kaspa node](/integrate/kaspa-node) when you are ready to run your own backend.
